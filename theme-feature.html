Wordpress Theme

I developed this Wordpress theme to be used as a template for a portfolio. I began with a static portfolio page that I has built and from there converted it to a theme and added some functionalty. This theme is meant to be very simple and easy to navigate. I chose not to inlcude a lot of widget areas or complicated layouts as this is meant to be a very minimalist and porfessional design. This theme includes support for custom menus, customizable header elements, and thumbnails with custom sized. I used two plugins to develop this theme include CustomPostTypesUI and customfieldsUI in order to create custom templates for porftio pieces. The theme also inlcudes custom page types. I designes a set of user icons using Sketch to be used in the custom header as a logo for the user. I chose to create a Wordpesss theme because Worpress,as od 2016, powers 50% of the internet. While I was previously comfortable working with Wordpress and customizing content, I wanted to gain more experience with in-depth customization adn to sharpen my php skills. I've included this project in my portfolio because I think it displays my ability to create friendly and simple UI design content for Wordpress sites.
